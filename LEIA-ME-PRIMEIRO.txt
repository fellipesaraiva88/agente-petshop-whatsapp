🎬 RESUMO DO PROCESSO:

TERMINAL 1:
  $ cd /Users/saraiva/agentedaauzap
  $ ./instalar-tudo.sh
  ✅ Servidor rodando! [DEIXE ABERTO]

TERMINAL 2 (nova aba - Cmd+T):
  $ npx ngrok http 3000
  📋 Copie a URL: https://abc123.ngrok.io
  [DEIXE ABERTO]

TERMINAL 2 (mesmo terminal):
  $ cd /Users/saraiva/agentedaauzap
  $ ./scripts/configure-webhook.sh https://abc123.ngrok.io/webhook
  ✅ Webhook configurado!

CELULAR:
  Envie mensagem no WhatsApp
  Aguarde 5-10 segundos
  🎉 BOT RESPONDE!


💡 DICAS:

• Mantenha os 2 terminais abertos enquanto usar
• A URL do ngrok muda toda vez (precisa reconfigurar webhook)
• Para produção, use servidor real (não ngrok)
• Customize em: src/services/OpenAIService.ts
